# Instaclone
## InstaClone-mobile
노마드코더의 [풀스택]인스타그램 클론코딩 강의를 들으며 진행한 프로젝트입니다.

## 기술 스택
- react-native CLI
- typescript
- apollo client
- styled component

## 구현 범위
- Log In
- Sign Up
- Feed (using infinite scroll)
- Upload Photo
  - camera 및 album 연동
- Delete Photo
- Like / Unlike Photo
- Create / Delete Comments
- Search Photo
- Profile Page
- Follow / Unfollow User
- Direct Message(using subscription)
